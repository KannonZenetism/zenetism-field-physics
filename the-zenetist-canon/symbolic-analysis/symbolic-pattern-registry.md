## Methodological Principles

1. **Symbols carry structural function, not arbitrary meaning.** Bread in scripture functions as Orienting Logos nourishment because the textual tradition uses it consistently as such. Symbolic readings are constrained by traditional usage, not freely assigned.
2. **Surface object ≠ structural function.** The bread is not loaves; the fish is not protein; the wilderness is not desert. Literal-only reading misses the function the text is performing.
3. **Cross-traditional symbol clusters reveal common structural operations.** Bread / manna / soma / amrita / ambrosia all function as nourishment-from-Source across traditions. The cluster reveals a structural function (Source-nourishment) that the local symbol articulates culturally.
4. **Symbols can be centropic, entropic, or bifurcal depending on operative context.** Blood symbolizes life-force in some contexts, ritual binding in others, structural correction in others. Context determines orientation.
5. **Cascade and conflation principles apply to symbols as to figures.** A single symbolic current can manifest across layers (centropic cascade) or be culturally compressed across structurally distinct functions (conflation).
6. **Logos must be qualified by layer.** Per the Note on Logos Disambiguation in the Mythic Figure Layer Registry, bare "Logos" references should be replaced with the layer-qualified term: Orienting Logos (L₅ Theon — coherence as direction), Structuring Logos (L₄ DL Sophis — coherence as pattern), Christos Incarnate or Embodied Reason (L₁), Inverse Logos (IL₄ Nyxea — structural mimicry without coherence). Symbolic entries citing Logos function specify which layer the symbol carries.

## Symbolic Operators contra Symbolic Carriers

A symbol is not the object itself but the structural operation it performs. The object is the carrier; the operation is the operator.

Bread carries Orienting Logos nourishment.
Water carries purification.
Oil carries consecration.
Blood carries entropic life-force or covenant-binding depending on context.

The registry maps operators, not carriers.

A single carrier can host multiple operators depending on textual or ritual context. A single operator can appear through multiple carriers across traditions.

This distinction prevents symbolic conflation and preserves the Non-Fusion Axiom: same pattern, distinct instance; recurrence without merger.

## Hypostasis / Symbolic Narrative Distinction

Per the Note on the Hypostasis / Mythic Narrative Distinction in the Mythic Figure Layer Registry, the same diagnostic frame applies to symbols as to figures.

**Symbolic operators are:** structural functions, non-anthropomorphic, non-psychological, non-narrative.

**Symbolic narratives are:** anthropomorphic dramas, culturally refracted, moralized and politicized, using human or naturalistic motives to dramatize cosmic patterns.

When narratives assign anthropomorphic behaviors to symbolic elements — the cup that betrays, the mirror that lies, the serpent that tempts — orientation conflation occurs at the symbolic level. The serpent's "temptation" of Eve is narrative dramatization of an IL₄ Nyxea structural function (counterfeit knowledge offered as original); the serpent does not psychologically tempt in the structural reading. The mirror does not "lie"; it carries Khaonic emptiness that mimicry exploits when discernment is absent.

Parsing requires distinguishing the symbol's structural operation from the narrative drama imposed on it. The symbol carries the operation regardless of whether the narrative anthropomorphizes that operation as motive, intent, or behavior.
