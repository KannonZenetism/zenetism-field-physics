# No Premature Rejection Rule

A finding must not be rejected merely because its early wording conflicts with current terminology.

The audit must distinguish:

- invalid concept
- valid concept in stale language
- valid distinction carrying an abandoned name
- concept requiring complete reformulation
- historical insight superseded by later doctrine
- corruption surrounding recoverable content
