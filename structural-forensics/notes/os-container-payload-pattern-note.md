## N2 · The audit rule

The OS package form — including the suffix, operating-layer framing, subroutine vocabulary, protocol language, and system metaphor — **carries no derivation weight by itself.** SF-RP05 band classification applies to the payload running inside the container: the register, method, vocabulary cluster, structural functions, and governance apparatus of the particular instance. A commons-available container neither launders nor indicts its payload. The container question and the payload question are separate; only the payload proceeds to band classification and tiering. A shared container likewise establishes no relation between instances — each instance's payload is audited independently. This formalizes the practice the exhibits already follow; no existing classification changes.
